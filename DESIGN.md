# VTT Careers Design System

## Typography
- Primary Font: Inter, sans-serif
- Headings: Bold, clean, modern

## Colors
- Primary: #F97316 (Orange - for CTAs, buttons, active states)
- Secondary: #10B981 (Green - for success, hiring, active jobs)
- Background: #F8FAFC (Light slate for public), #0F172A (Dark slate for dashboard cards/sidebar)
- Text: #334155 (Slate 700)
- Muted: #64748B (Slate 500)

## UI Elements
- Style: Glassmorphism
- Corners: Rounded-xl (12px)
- Shadows: Soft, diffuse shadows (shadow-lg)
- Effects: backdrop-blur-md for glass panels
- Borders: border border-white/20 (for glass effect)

## Layout
- Sidebar: 240px wide, fixed, glassmorphism background
- Topbar: 64px high, sticky, glassmorphism
- Content Area: Padded, responsive grid

## Animations
- Pulse effect on primary CTAs
- Smooth hover transitions (duration-300)
- Laser scan effect for AI processing areas (simulated via gradients)
